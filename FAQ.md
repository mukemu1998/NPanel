# 常见问题

## 1. 为什么远程登录返回 `503`？

通常是以下原因之一：

- 没有配置 `ADMIN_PASSWORD`
- 没有配置 `SESSION_SECRET`
- 当前部署实例没有绑定 D1

## 2. 为什么 `/api/health` 显示 `mock`？

说明当前实例运行时没有绑定 D1，使用的是本地 mock 存储模式。

## 3. 真实代理流量可以走 Cloudflare 吗？

不建议。

这个项目的设计目标是管理节点和分发订阅，不是承载真实代理流量。实际代理流量应直接连接自己的 VPS 或中转入口。

## 4. 演示数据适合生产环境吗？

不适合。

演示数据只用于示例和测试，不适合作为真实生产数据使用。

## 5. 默认品牌名可以修改吗？

可以。

可以在登录后的面板设置里修改，也可以直接编辑 `public/panel-config.js` 中的默认值。

## 6. 为什么有些 Clash 客户端仍然显示自己的导入名？

因为部分客户端，尤其是 URL 导入场景，会自己决定本地配置名称显示方式。

服务端可以控制订阅内容和响应头，但无法完全统一所有客户端的本地显示逻辑。
